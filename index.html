「Projects」

【Alipoodle.me】

【Welcoming Bot for Discord】
This is a bot for Discord made in Javascript, that will allow you to have a room where a user will have to agree to the rules of a server by sending a message and, will then give the user a role and welcome them in a chosen room. This bot will also remove Discord Links and post a message in either the room or a bot-log room.

【osu Info】
This is a program for Windows that shows you the information about your osu account in a nice clean Material design. Allows for Dark and Light themes and a choice of Accent colour (Made with GreenTurtwig ). This shows some of your general information: such as Rank, As, Ss, SSs and PP, Top 5 plays and Recent Plays both, in a neat table.

【GPMDP Rainmeter Player】
This is a Skin for Rainmeter, a background customers, that will show the information of the current song you are listening to on Google Play Music Desktop Player. There's various Variables used in it for easy change of how colours and font.

【GPMDP Midnight Accent】
This is a CSS File that can be added to Google Play Music Desktop Player to allow you to use the dark theme but have the top bar, border and a few other things with the accent colour similar to how it is on Light theme. I'm always keeping this up-to date, as I love Google Play Music so much! This theme is based off Play Midnight

【BetterDiscord Dark Matter Update】
This is an updated version of the BetterDiscord theme Dark Matter by cosmicsalad. This themes the new settings and has an Additional theme for adding Material Icons. I will continue to do updates for this as the main developer seems to have give up on it.
