【Welcoming Bot for Discord】
This is a bot for Discord made in Javascript, that will allow you to have a room where a user will have to agree to the rules of a server by sending a message and, will then give a role and welcome the user in a chosen room.

【osu Info】
This is a program for Windows that shows you the information about your osu account in a nice clean Material design. Allows for Dark and Light themes and a choice of Accent colour (Made with GreenTurtwig)

【GPMDP Rainmeter Player】
This is a Skin for Rainmeter that will show the information of the current song you are listening to on Google Play Music Desktop Player

【GPMDP Midnight Accent】
This is a CSS File that can be added to Google Play Music Desktop Player to allow you to use the dark theme but have the top bar, border and a few other things with the accent colour you pick.

【BetterDiscord Dark Matter Update】
This is an updated version of the BetterDiscord theme Dark Matter, This Themes the new settings and has an Additional theme for adding Material Icons.
